# Discord 表情統計機器人
由於Discord的客制表情是有上限的，故生了一個機器人來計算表情符號

## Getting Started
- Visit discord developer website to create a discord bot accout
- Create a `.env` file and enter your token like this example below
```
TOKEN = "enter your token"
```
